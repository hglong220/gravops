# 🚀 快速安装指南

## 安装步骤（3分钟搞定）

### 1️⃣ 打开 Chrome 扩展管理
在 Chrome 浏览器地址栏输入并回车：
```
chrome://extensions/
```

### 2️⃣ 启用开发者模式
在页面右上角，打开 **"开发者模式"** 开关

### 3️⃣ 加载扩展
1. 点击左上角 **"加载已解压的扩展程序"**
2. 浏览并选择文件夹：
   ```
   g:\gravops\zcy-goldminer
   ```
3. 点击 **"选择文件夹"**

### 4️⃣ 验证安装
看到 **"政采云核心类目挖掘机 v1.0"** 就成功了！

---

## 开始使用

### 第一步：登录政采云
访问 https://www.zcygov.cn/ 并登录你的账号

### 第二步：进入类目选择页面
导航到：**商品发布 > 类目选择** 页面  
或直接访问：https://www.zcygov.cn/goods-center/goods/category/attr/select

### 第三步：启动挖掘
1. 等待右下角出现 **"⛏️ 类目挖掘机"** 面板（约2秒）
2. 点击 **"2. 开始挖掘核心数据"**
3. 等待完成（通常需要 30-60 秒）
4. 自动下载 `zcy_core_categories.json` 文件

---

## ⚠️ 常见问题

**Q: 面板没有出现？**  
A: 刷新页面，确保你在 *.zcygov.cn 域名下

**Q: 挖掘失败？**  
A: 检查是否登录，打开 F12 查看 Console 错误信息

**Q: 数据不完整？**  
A: 先点击 "1. 扫描左侧菜单" 更新 ID，再开始挖掘
