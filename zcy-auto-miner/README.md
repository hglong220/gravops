# 政采云类目自动提取工具 (Playwright版)

## 🎯 方案说明

通过 **Playwright 浏览器自动化**，模拟正常的供应商发布商品流程，**100% 合法**地提取政采云全部类目数据。

### ✨ 核心优势

- ✅ **完全合法** - 使用真实供应商账号登录，通过正常流程操作
- ✅ **100%准确** - 直接从政采云官方页面提取，数据与官方完全一致
- ✅ **零风控风险** - 模拟真实用户行为，不会触发反爬或封号
- ✅ **多格式输出** - JSON（扁平+树形）、TXT、CSV 自动生成
- ✅ **自动化程度高** - 只需登录一次，其余全自动完成

## 🚀 快速开始

### 方法一：双击运行（推荐）
```
双击 启动.bat
```
根据提示选择运行模式即可！

### 方法二：命令行运行
```bash
# 1. 安装依赖
npm install

# 2. 安装浏览器
npm run install-browser

# 3. 运行提取
npm run extract
```

## 📋 使用流程

1. **启动脚本** → 自动打开浏览器
2. **手动登录** → 输入政采云账号密码
3. **自动导航** → 脚本自动跳转到商品发布页面
4. **自动提取** → 自动展开类目并提取数据
5. **自动保存** → 生成 JSON/TXT/CSV 文件

## 📂 输出文件

所有文件保存在 `./output` 目录：

| 文件 | 用途 |
|------|------|
| `categories_flat.json` | 扁平列表，适合数据库导入 |
| `categories_tree.json` | 树形结构，适合前端渲染 |
| `categories.txt` | 制表符分隔，适合Excel |
| `categories.csv` | 标准CSV格式 |

## 🔧 技术栈

- **Playwright** - 浏览器自动化
- **Node.js** - 运行时环境
- **fs-extra** - 文件操作

2. 首次运行需要下载 Chromium（约 200MB）
